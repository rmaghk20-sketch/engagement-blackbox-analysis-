# Opening the Black Box of Behavioral Engagement

Analysis code for the manuscript "Opening the Black Box of Behavioral 
Engagement: Explainable Machine Learning Evidence from 50,000 
Block-Based Programming Projects" (under review).

## Data
Entry Research Educational Dataset (NAVER Connect Foundation).
Available from the provider under a research data-use agreement:
research@connect.or.kr

## Requirements
Python 3.10+, pandas, scikit-learn, xgboost, shap

## Usage
Place the agreement-provided CSV in the working directory and run:
python analysis_public.py
